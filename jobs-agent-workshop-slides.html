Workshop

Use jobs-agent with the AI assistant you already know

We will install jobs-agent by pasting a GitHub link into ChatGPT, Claude, Codex, or Claude Code, then use plain English to search, tailor, apply, and follow up.

Goal: leave with a repeatable job-search routine, not a list of commands. 10 slides
Start by reducing anxiety: the participant does not need to memorize commands. The workshop teaches what to ask the assistant to do.

Step 1

Install jobs-agent by asking the assistant

Copy, paste, and send this

Please install jobs-agent from this GitHub repo and set it up for this workspace:
https://github.com/hassanshabbirahmed/jobs-agent

Use ChatGPT, Claude, Codex, or Claude Code. If it cannot install tools, ask the instructor for an agent-enabled workspace.

Then ask: "Tell me when it is ready, and explain the next step in simple language."

Have your resume ready before the next slide. Install first
Give participants time to paste the link. If a tool cannot install, pair them with the prepared demo workspace.

Step 2

Ask jobs-agent to build your job-search profile

Say this to the assistant

Help me set up jobs-agent. Use my resume, ask me for my job preferences, and interview me about my work history one question at a time.

  • Resume: upload the file when asked.
  • Preferences: role, location, remote rules, salary, industries.
  • Work history: tell real stories, wins, tools, and responsibilities.
Better setup means better matches, resumes, and letters. One-time setup
Encourage concrete answers. "Managed customers" is weaker than "managed 40 enterprise accounts and reduced churn."

Step 3

Search for jobs with normal language

Start broad

Find jobs that match my resume and preferences. Show me the best matches first and explain why each one fits.

Then narrow it

Search again, but focus on remote customer success roles at B2B software companies posted in the last two weeks.

You can change the role, industry, location, or seniority in plain English. Search like a conversation
Demonstrate how changing one sentence changes the search. No one needs to know a command name.

Step 4

Turn search results into a shortlist

1

Ask for a simple table

"Make a shortlist with company, role, fit score, why it fits, and any concerns."

2

Remove dealbreakers

"Remove roles that require relocation, a visa I do not have, or skills I clearly lack."

3

Pick two or three

"Choose the top three jobs worth tailoring for today."

The agent saves time; you still choose where to spend your energy. Quality over quantity
This is where participants learn not to apply everywhere. A shortlist protects their attention.

Step 5

Tailor your resume for one selected job

Paste a job link and ask

Tailor my resume for this job posting. Keep it honest, preserve my real experience, and explain what you changed.

Review these details

Company name, job title, dates, metrics, tools, skills, and any claim that sounds stronger than your real experience.

If something feels off, say: "Make this more accurate and less exaggerated."

A tailored resume should be specific, not fictional. Truth first
Stress that AI should not invent credentials. The participant is accountable for the final resume.

Step 6

Draft a cover letter, then make it sound like you

Ask for a first draft

Write a concise cover letter for this job using my tailored resume. Make it specific to the company and avoid generic phrases.

Then personalize it

Make the tone warmer and simpler. Keep it under one page and make the opening sound less formal.

Good letters sound like a person who understands the role. Edit by asking
Show one quick revision prompt. Participants should see that they can steer tone without technical knowledge.

Step 7

Apply with the agent, but check every field

Say this before applying

Help me fill this application using the tailored resume and cover letter. Stop before submitting so I can review everything.

  • Check: name, email, phone, location, and links.
  • Check: salary, work authorization, and availability.
  • Check: uploaded files and custom questions.
The most important button is not "submit"; it is "review." Human approval
Make this a safety rule. The agent can assist, but participants should never blindly submit.

Step 8

Track applications and ask for follow-up help

Track the basics

Company, role, date applied, link, status, and next action.

Use your network

Ask jobs-agent to check contacts or help draft a polite intro request.

Follow up calmly

After a few days, ask for a short follow-up message that sounds natural.

Applications disappear when they are not tracked. Keep one simple list. Do not lose leads
This slide is about habit. The agent helps participants keep momentum after the workshop.

Practice Loop

The repeatable routine

1

Ask for matching jobs

Use a natural sentence with role, location, industry, and must-haves.

2

Choose one job and tailor

Ask for a tailored resume, review it, then ask for a cover letter.

3

Apply, track, follow up

Let the agent help with the form, stop before submission, then log the application.

AI agent speed plus human judgment is the whole method. Repeat weekly
End with a live exercise: everyone picks one job and writes the next prompt they will send.
